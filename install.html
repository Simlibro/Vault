Installation

Install Vault as an app

For the most reliable desktop installation, use Chrome, Edge, or Brave. Firefox can still use Vault in a normal browser tab, but installed PWA support is less reliable.

Desktop: Chrome, Edge, Brave

  1. Open Vault by visiting app.html.
  2. Use the browser install option from the address bar or browser menu.
  3. Open the installed app so Vault runs in its own window with less browser chrome.

Firefox desktop

  1. Use Vault in the browser if you prefer Firefox.
  2. Do not depend on Firefox for installation if you want a true standalone app window.
  3. Use Chrome, Edge, or Brave for the install step if needed.

Android

  1. Open Vault in Chrome, Edge, or Brave.
  2. Choose Install app or Add to Home screen from the prompt or browser menu.
  3. Launch from the home screen as a standalone app.

iPhone and iPad

  1. Open Vault in Safari.
  2. Tap Share.
  3. Choose Add to Home Screen.

If install does not appear

Open Open Vault directly, refresh once, and check the browser menu. If an older service worker or installed copy is interfering, clear site data or remove the old installed version and try again.

Vault can always be used in a normal browser tab, even when installation is unavailable.
