# Vault

Vault is a small offline password manager Progressive Web App.

The goal of Vault is simplicity and transparency.

Passwords are encrypted locally in the browser using the Web Crypto API and stored on the user's device. The application does not require an account and does not send password data to any server.

## Features

- Local encryption using the browser
- No cloud account
- No remote password storage
- Manual encrypted export and import
- Runs offline as a PWA
